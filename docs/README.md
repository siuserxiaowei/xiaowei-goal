# Docs Archive

This directory is for GitHub Pages output and historical research/report
artifacts.

It is not part of the core xiaowei-goal skill runtime. Agents should treat the
current skill behavior as defined by:

- `SKILL.md`
- `references/`
- `examples/`
- `agents/interface.yaml`
- `manifest.json`
- `scripts/`

Do not infer current skill rules from old reports in this directory unless a
task explicitly asks to inspect the archived pages.
