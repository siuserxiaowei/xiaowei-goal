# Domain Packs

Use this reference after the general task pack. Domain packs make goals more
specific to common Xiaowei work.

Choose at most one primary domain pack unless the user asks for a combined
strategy.

## AI Tool Site Pack

Use for AI utility sites, free tools, generators, converters, small SaaS tools,
and SEO-led AI products.

Require mapping research to:

- first-viewport promise
- demo or try-it path
- free tool entry
- SEO page opportunities
- activation action
- trust proof
- conversion path
- implementation goal

## Chrome Extension Pack

Use for Chrome extensions and browser workflow products.

Require mapping research to:

- Chrome Web Store title and description
- permission explanation
- first install experience
- browser-in-context workflow
- privacy risk
- review risk
- retention trigger
- implementation goal

## Global SaaS Pack

Use for outbound/global SaaS, English-first products, multi-region landing
pages, or tools targeting overseas users.

Require mapping research to:

- target country, language, and user segment
- positioning and pricing packaging
- competitor pages
- SEO, directory, community, and launch channels
- onboarding
- trust, compliance, refund, and payment risks
- implementation goal

## Xiaohongshu / Douyin Validation Pack

Use for validating demand, hooks, content angles, and user language from
Xiaohongshu, Douyin, or adjacent short-form/social platforms.

Require mapping research to:

- content hook
- user pain
- comment-section signal
- viral structure
- low-cost validation action
- what must not be copied
- next experiment goal

## GitHub Open Source Growth Pack

Use for developer tools, open-source projects, GitHub-first distribution, and
repository growth.

Require mapping research to:

- README first screen
- demo or gif
- examples and quickstart
- issue/discussion templates
- stars and contributor path
- launch channels
- developer trust
- next growth goal

## Domain Pack Field

Generated goals should include:

```text
领域包：[AI 工具站包 / Chrome 插件包 / 出海 SaaS 包 / 小红书/抖音内容验证包 / GitHub 开源项目增长包 / 不适用]；本次必须把研究或执行结果映射到[domain-specific outputs]。
```
